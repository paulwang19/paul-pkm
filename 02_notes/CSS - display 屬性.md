# 簡介

`display` 是最基本 CSS style 元素屬性，決定了一個元素的盒子模型類型 (box model type)，意即這個元素在頁面上怎麼被呈現、怎麼佔據空間、以及如何與周圍元素互動

# 常用值

| 屬性值   | 效果     | 常見用途                    |
| -------- | ---------- | ------------------------- |
| `block` | 像一個區塊，會佔據一整行 | `<div>`, `<p>`, `<h1>` - `<h6>`, `<form>`, `<ul>`, `<li>` |
| `inline` | 像文字一樣，與其他行內元素在同一行內流動，寬度由內容撐開 | `<a>`, `<span>`, `<strong>`, `<img>` |
| `inline-block` | `inline` 和 `block` 的混合體，不會佔據一整行，且可以設定 margin 和 padding | 並排的導覽列按鈕或卡片 |
| `none` | 元素從頁面上完全消失 | 開合式選單等隱藏內容 |
| `flex` | 現代排版模型 | 導覽列、側邊欄或任何需要彈性對齊的元件及佈局 |

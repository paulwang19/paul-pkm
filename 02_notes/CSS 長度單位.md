# 前言

HTML 元素中 CSS style 屬性常會用到 `width` 或 `height`，例如 : `width: 100%`，這邊用到的 `%` 及是一個長度單位，此筆記介紹長度可用的單位

# 單位

分成**絕對單位**和**相對單位**

## 絕對單位

| **單位**   | **名稱**     | **說明**                    |
| -------- | ---------- | ------------------------- |
| **`px`** | 像素 (Pixel) | **最常用**的絕對單位。它代表螢幕上的一個「點」 |
| **`pt`** | 點 (Point)  | 1 pt = 1/72 英吋。主要用於印刷排版   |
| **`pc`** | Pica       | 1 pc = 12 pt。主要用於印刷排版     |
| **`cm`** | 公分         | Centimeter                |
| **`mm`** | 公釐         | Millimeter                |
| **`in`** | 英吋         | Inch                      |

## 相對單位

| **單位**     | **名稱**           | **說明**                                                        |
| ---------- | ---------------- | ------------------------------------------------------------- |
| **`em`**   | Em               | **相對於當前元素的字體大小**。如果用在 `font-size` 上，則是相對於 _父元素_ 的字體大小         |
| **`rem`**  | Root Em          | **相對於根元素（`<html>`）的字體大小**。這是目前最推薦用於設置字體和間距的單位，因為它提供了全局一致的縮放基準 |
| **`vw`**   | Viewport Width   | **相對於可視區域（Viewport）的寬度**。`100vw` 等於瀏覽器視窗的 100% 寬度             |
| **`vh`**   | Viewport Height  | **相對於可視區域的高度**。`100vh` 等於瀏覽器視窗的 100% 高度                       |
| **`vmin`** | Viewport Minimum | `vw` 和 `vh` 中 _較小_ 的那個值                                       |
| **`vmax`** | Viewport Maximum | `vw` 和 `vh` 中 _較大_ 的那個值                                       |
| **`ch`**   | Character        | 相對於當前字體中「0」字元的寬度                                              |
| **`ex`**   | X-height         | 相對於當前字體中「x」字母的高度                                              |

# 常見問題

> [!question]
> `100%` 含意是甚麼

分成寬度和高度兩個部分討論

## 寬度

如前言，`width: 100%` 意味著其父元素 100% 的寬度值，如果父元素寬度是 500 px，則這個元素寬度就是 500 px

一般情況下，寬度只會計算「內容寬度」，即不含 padding 和 border（較不直觀），通常採用較直觀的作法是加上 `box-sizing: border-box`，讓寬度自動包含 padding 和 border

## 高度

這是常見陷阱⚠️，`height: 100%` 是父元素高度的 100%，但父元素高度未指定前，預設會使用 `height: auto`，即隨著實際內容高度來計算這個元素高度。

沒有明確高度的父元素，當前元素 100% 的 `auto` 是無法計算的，`height: 100%` 在這種時候會失效。

唯一讓 `height: 100%` 生效的情況是，確保所有的祖先元素（一路到 `<html>`）都有可被計算的高度

> 造成寬度和高度有如此差異的原因，是因為[[瀏覽器渲染 CSS 決定寬度與高度的基本原理|瀏覽器對於寬度與高度渲染原理不同]]

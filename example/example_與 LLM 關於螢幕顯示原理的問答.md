# 學習摘要

今天向 LLM 詢問了電腦螢幕的顯示原理，獲得了幾個關鍵知識點：

1.  **基本單位是「像素」 (Pixel):** 螢幕由無數個像素點組成，每個像素點又包含紅 (R)、綠 (G)、藍 (B) 三個「子像素」。透過控制這三個子像素的亮度，就能混合出各種顏色。

2.  **主流技術是 LCD (液晶顯示):** 它的結構像三明治，關鍵在於「液晶」層。它本身不發光，依賴後方的「背光模組」。液晶分子透過電壓控制其扭轉角度，來決定讓多少光線通過，再經過彩色濾光片形成顏色。

3.  **新興技術是 OLED (有機發光二極體):** 它不需要背光模組，因為每個子像素自己就是一個能發光的燈泡。當需要顯示黑色時，像素可以直接「關燈」，所以黑色特別純粹，對比度極高。

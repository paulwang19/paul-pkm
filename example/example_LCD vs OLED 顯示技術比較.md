# 連結的概念

- [[LCD 液晶螢幕原理]]
- [[OLED 自發光螢幕原理]]

# 比較表格

| 特性 | **LCD (液晶顯示)** | **OLED (有機發光二極體)** |
| :--- | :--- | :--- |
| **發光原理** | 依賴背光模組，穿透式 | 像素自發光，發射式 |
| **黑色表現** | 輕微漏光，呈深灰色 | 像素關閉，實現「真黑」 |
| **對比度** | 較低 | 極高 (近乎無限) |
| **反應速度** | 較慢 | 極快 |
| **厚度** | 較厚 (因有背光層) | 非常薄 |
| **潛在問題** | 漏光、可視角度較差 | 成本較高、有機材料壽命 (烙印) |

# 總結

OLED 在畫質的關鍵指標（如對比度、黑色表現）上全面超越 LCD，但 LCD 憑藉其成熟的技術和成本優勢，目前仍在市場上佔有重要地位。選擇哪種技術取決於產品的定位和成本考量。
